# Flutter Chat Application

A simple and practical **Real-time** chat application built using **Flutter** and **Supabase**. This project focuses on learning how modern mobile apps handle authentication, real-time data, cloud storage, and secure configuration.

The app allows users to create an account, log in, upload a profile image, and chat with other users in real time.

---

## 📌 Purpose of This Project

This project was created to:

* Understand how backend services like Supabase work with Flutter
* Learn user authentication and session management
* Implement real-time messaging
* Practice clean project structure
* Follow industry-standard security practices
* Build a portfolio-ready mobile application

It is mainly intended for learning, practice, and showcasing Flutter development skills.

---

## 🚀 Features

* User authentication (Sign up / Login)
* Secure session handling
* Real-time chat messages
* Profile image upload
* Cloud database storage
* Automatic login persistence
* Clean and responsive UI
* Environment-based configuration

---

## 🛠️ Tech Stack

* **Frontend:** Flutter (Dart)
* **Backend:** Supabase
* **Authentication:** Supabase Auth
* **Database:** Supabase PostgreSQL
* **Storage:** Supabase Storage
* **State Management:** Flutter default state handling
* **UI:** Material 3
